Este repositório reúne todo o conteúdo utilizado ao longo das aulas, incluindo notebooks, exercícios, respostas e atividades práticas.

##  Estrutura do Repositório

##  Objetivo do Curso

O curso tem como objetivo desenvolver as habilidades necessárias para trabalhar com Inteligência Artificial, abordando:

- Lógica de programação com Python  
- Estruturas de dados  
- Manipulação de dados com Pandas e NumPy  
- Leitura e tratamento de dados  
- Introdução a Machine Learning  
- Aplicações práticas em IA  

---

##  Como Utilizar Este Repositório

Para obter o máximo aproveitamento:

1. Acompanhe os notebooks durante as aulas  
2. Tente resolver os exercícios antes de consultar as respostas  
3. Utilize os desafios para aprofundar o aprendizado  
4. Execute os códigos em:
   - Jupyter Notebook  
   - Google Colab  

---

##  Importante

- Este material é de uso educacional  
- Evite copiar respostas sem tentar resolver  
- Traga dúvidas para discussão em aula
